Контекст:
Реализовать возможность забирать значения из файла valid_addresses.txt и направлять их в файл, расположенный по адресу http://homebase.ru/"Наименование файла"

Условия реализации:
1. После забора сведений необходимо формировать отдельный файл в репозитории в виде таблицы со столбцами region, city, street_address, postcode.
Причем street_address должен содержать как улицу, так и дом.
2. Сортировка в этой таблице должна начинаться с region, потом city, потом street_address, потом postcode в порядке возрастания в алфавитном порядке;
3. Сортировка внутри столбцов должна быть в порядке возрастания по алфавиту и по типам.
4. Таблица должна сохранять только валидные по структуре адреса.
